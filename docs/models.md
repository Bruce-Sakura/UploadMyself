# 模型说明

## 语音克隆

### GPT-SoVITS v2
- **类型**: 少样本语音克隆 + TTS
- **输入**: 3-10 分钟参考音频 + 文本
- **输出**: 克隆声音的语音
- **优势**: 中文效果好，few-shot 能力强
- **许可**: MIT

### CosyVoice2
- **类型**: 多语言语音合成
- **输入**: 5 秒参考音频 + 文本
- **输出**: 克隆声音
- **优势**: 多语言支持，阿里出品
- **许可**: Apache 2.0

## 2D 数字人

### LivePortrait
- **类型**: 人脸动画驱动
- **输入**: 源图像 + 驱动视频/音频
- **输出**: 动态说话视频
- **优势**: 高质量表情迁移，实时性好
- **许可**: MIT

### SadTalker
- **类型**: 音频驱动人脸动画
- **输入**: 图片 + 音频
- **输出**: 说话视频
- **优势**: 学术界标杆，口型同步好
- **许可**: MIT

## 3D 数字人

### InstantMesh
- **类型**: 单图/多图 3D 重建
- **输入**: 1-4 张图片
- **输出**: 3D Mesh (.obj/.glb)
- **优势**: 速度快，质量高
- **许可**: Apache 2.0

### 3D Gaussian Splatting
- **类型**: 神经辐射场渲染
- **输入**: 多角度图片
- **输出**: 实时渲染的 3D 场景
- **优势**: 实时渲染，质量极高
- **许可**: 学术许可

## 模型蒸馏

支持将上述大模型蒸馏为轻量版本：
- LLM: 72B → 7B (知识蒸馏)
- Voice: Large → Small (特征蒸馏)
- Avatar: Base → Lite (结构化剪枝 + 蒸馏)
